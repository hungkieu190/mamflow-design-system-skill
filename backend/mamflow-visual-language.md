# MamFlow Visual Language

## Philosophy

The interface should feel:

* Professional
* Trustworthy
* Technical
* Calm

Not:

* Playful
* Trendy
* Corporate Enterprise
* Startup Marketing

---

# Color System

## Primary

Purpose:

Primary actions only.

Color:

#2563EB

Usage:

* Primary button
* Selected state
* Active tab
* Focus state

Do not use primary color for decoration.

---

## Neutral Scale

Background:
#FFFFFF

Secondary Background:
#F8FAFC

Border:
#E2E8F0

Divider:
#CBD5E1

---

## Text

Primary:
#0F172A

Secondary:
#475569

Muted:
#64748B

Disabled:
#94A3B8

---

## Status Colors

Success:
#16A34A

Warning:
#F59E0B

Error:
#DC2626

Info:
#2563EB

No additional status colors.

---

# Color Usage Rules

80% Neutral

15% Primary

5% Status Colors

Avoid colorful interfaces.

Color should communicate meaning.

Not decoration.

---

# Buttons

Primary:
Solid Primary Color

Secondary:
White + Border

Danger:
Red Text

Never create additional button variants.

---

# Cards

White Background

Light Border

Minimal Shadow

Cards should feel stable.

Not floating.

---

# Charts

Use:

Primary
Success
Warning
Error

Maximum 4 colors per chart.

Avoid rainbow charts.

---

# Icons

Default:
Text Secondary Color

Status icons:
Use corresponding status color.

Do not color icons unnecessarily.

---

# Accessibility

Minimum contrast:

WCAG AA

Always prioritize readability over aesthetics.
